# Manjit Pun — Noir Terminal Portfolio

A static portfolio built with semantic HTML, modern CSS, and vanilla JavaScript. It uses a responsive terminal-inspired UI, accessible theme switching, project/certificate previews, and local assets.

## Run locally

No installation step is required. From this directory, run one of the following:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.

For a production-ready static deployment, upload the contents of this directory to any static host (GitHub Pages, Netlify, Vercel static hosting, Cloudflare Pages, etc.). There is no transpilation or bundling command because this project has no package manifest or build tool.

## Main files

- `index.html` — semantic structure and portfolio content
- `styles.css` — responsive noir/terminal visual system and theme tokens
- `script.js` — theme persistence, terminal commands, animations, navigation, dialog controls
- `images/` — project, skill, certificate, and profile images
- `files/ManjitPun.pdf` — downloadable résumé
